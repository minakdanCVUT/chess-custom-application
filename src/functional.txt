software package:

    configuration package:

        1) ChessCellColor - enum type для хранений цвета клетки на доске
        2) ChessFigureColor - enum type для хранений цвета фигуры на доске
        3) ChessPosition - содержит координаты фигуры или клетки на доске
        4) JavaFXButtonStorage - класс для хранения информации внутри JavaFX кнопки,
        чтобы не таскать информацию как параметры в функции, в определенный момент
        времени создается обьект storage, который содержит фигуру, которая стоит(?) на клетке(кнопке)

    domain package:

        1) figures package - содержит все основные фигуры с переопределенными методами
        из абстрактного класса ChessFigure
        2) heroes package - содержит все кастомные фигуры наследованные от класса ChessFigure
        и интерфейса ChessFigureHero

        3) ChessCell - класса контейнер клетки на доске, хранит информацию о фигуре цвете клетки
         ( близок к удалению из проекта из-за низкой используемости)
        4) ChessFigure - абстрактный класс фигуры со всеми нужными параметрами и методами,
        которые переопределены в дочерних классах
        5) ChessFigureHero - интерфейс для героев, содержит только один дополнительный метод skill()
        6) ChessMap - контейнер кнопок, клеток, настроек - основной(?) класс в игре


    service package - пока что не содержит методов, там будут определены методы для контроля правил игры

    start_config package:
        1) chess_figures.json - файл хранит иконки фигурок (в скором времени будет заменен на фотографии)
        2) start_config.json - содержит стартовые позиции фигур
        3) StartConfiguration - настройка всех стартовых фигур, создание кнопок, установка стилей, создание фигур
        определение фигур на кнопки
